# Libft
- 앞으로 유용하게 사용할 함수를 모아 **나만의 C 라이브러리**를 만드는 프로젝트다.
- 표준 라이브러리에 이미 존재하는 함수부터 새로운 함수까지 많은 양의 함수를 직접 구현해본다.     
- 기존 함수를 단순히 가져다 쓰기 전에 각 함수가 작동하는 방식을 깊이 있게 고민해 볼 수 있는 프로젝트였다.
